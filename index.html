In Artificial Intelligence we use models that are inspired by our human nervous system. They are called artificial neural networks. Each neural network consists of several layers of neurons. It starts with the input layer, then some so-called hidden layers and finally the output layer, which serves to derive the results.

In this exhibit you can write a number in the input field. The neural network then calculates which digit from 0 to 9 you most likely wrote. The colours of the individual pixels of the input image are converted into input values. The input image is 28x28 pixels, which results in 784 input values ​​for the neural network (28x28 = 784). A hidden layer with 100 neurons then follows in the network. At the end there are 10 output values ​​that indicate a likelihood. Each output represents a number between 0 and 9. For example, if the last output value is the highest of all numbers, the network calculated that you did most likely wrote a 9. If two output values ​​are the same, then the likelihood for both drawn digits is the same. In this case, the network cannot say whether you e.g. have drawn a 1 or a 7.

The individual input values ​​are weighted, i.e. multiplied by numbers, so-called weights. This increases or decreases the input values.

A positive or negative number, the so-called bias, is added to the sum of the input values. Then this calculated value is sent to a predefined function, a so-called activation function. It decides which value is passed on. In our case, the calculated value is passed on directly as output if it is positive. If it is negative, the value zero is passed on. In advanced mode you can try out different activation functions for the network. More details on the role of the weights and bias in learning can be found here.

In each neuron you find a predefined activation function and changeable values ​​(such as weights and bias). The changeable values ​​are particularly interesting for learning.

A new neural network does not “know” anything at the beginning. It has to learn or be trained first. For this purpose, the neural network is “fed” with training data.

In our case, training data consists of handwritten digits combined with information which digit it represents. The network now starts receiving individual training images as input. After each "individual training", the changeable values ​​(i.e. bias and weights) are updated so that the output of the network changes in the direction of the desired number - and not just for the current training picture, but for all training data. Then the network starts the next training session with the next training data. By continuously adjusting the bias and weights, the network gradually “learns” and the output data is becoming more and more accurate. This optimization method is called “gradient descent”. It is one of the most important methods of AI.

Training data is the basis for a neural network to do what you want. It is usually a large set of sample data combined with the desired outputs (= the solutions, so to speak). Often a large part of the training data (e.g. 80%) is used to train the network, while the remaining part (e.g. 20%) is used to test the network and to check its performance / accuracy. Training data is very valuable, and luckily there are open databases like the MNIST database of handwritten digits that we used for this exhibit.
