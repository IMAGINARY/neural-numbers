Les données d'entraînement sont fondamentales pour apprendre à un réseau de neurones à réaliser une tâche. Il s'agit d'habitude d'un vaste ensemble d'exemples de valeurs d'entrée, combinées avec les valeurs de sortie souhaitées. Souvent, une grande partie des données d'entraînement est utilisée pour entraîner le réseau, tandis que la partie restante est par la suite utilisée pour mettre à l'épreuve le réseau, en vérifiant son exactitude.

Les données d'entraînement sont très précieuses. Heureusement, il existe des bases de données ouvertes, comme la base de données de chiffres manuscrits MNIST dont nous nous sommes servis pour cette exposition.
