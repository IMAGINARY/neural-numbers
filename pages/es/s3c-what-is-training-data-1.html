Los datos de entrenamiento son fundamentales para enseñar a una red neuronal a realizar correctamente una tarea. Normalmente, consisten en una gran colección de ejemplos de datos de entrada, junto con sus correspondientes valores de salida deseados. Con frecuencia, se utiliza la mayor parte de los datos de entrenamiento para entrenar la red, mientras que se reserva otra parte para evaluar la red una vez entrenada, verificando su exactitud.

Los datos de entrenamiento son muy valiosos. Afortunadamente, existen bases de datos abiertas, como la base de datos de cifras manuscritas MNIST que hemos utilizado en esta exposición. Para el módulo Números Neuronales, hemos utilizado 70.000 imágenes de cifras manuscritas.
